# Tkinter 学习目录 (莫烦Python)
<img src="{{site.baseurl}}/static/img/course-cover/tkinter.jpg" height="200">

Tkinter 是 python 的一个跨平GUi 包. 他是一个制作窗口文件, 做模拟的最简单,也是最方便的模块. 不用下载安装其他的包, 也不用担心哪个系统是不是不兼容.

* 这套教材主要包含了:
 * 窗口的各种小部件;
 * 实际运用的一个登陆窗口的例子;
